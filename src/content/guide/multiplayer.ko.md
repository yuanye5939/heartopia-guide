Heartopia는 언제든지 최대 **11명의 다른 플레이어**가 보이는 공유 세계입니다. 이 가이드는 첫 친구 추가부터 함께 집 짓기까지 모든 멀티플레이어 기능을 다룹니다.

---

## 멀티플레이어 작동 방식

- 언제든지 최대 **11명의 다른 플레이어**가 당신의 세계에 보입니다
- 플레이어는 낚시, 원예, 탐험 등 각자의 활동을 하는 모습으로 나타납니다
- 채팅은 **확인된 친구끼리만** 가능 — 낯선 사람과의 강제 공개 채팅 없음
- 세상은 활기차지만 부담스럽지 않습니다

---

## Friendship Journal

**DG Level 4**에서 해금. 이것은 친구 목록이자 모든 소셜 기능의 허브입니다.

### 친구 추가 방법

1. 스마트워치에서 **Friendship Journal** 열기
2. 친구의 **UID**(고유 플레이어 ID) 입력
3. 친구 요청 전송 비용 **1 Wishing Star**
4. 수락되면 친구가 저널에 표시됨

### 친구 혜택

- 서로의 마을을 **무료**로 방문
- 친구가 오프라인일 때 **선물** 보내기 (우편함으로 전달됨)
- **Co-Build** 함께 하기 (DG Level 8+)
- 어디서나 자유로운 채팅

### UID 확인 방법

UID는 Friendship Journal과 스마트워치 설정에 표시됩니다. 친구들이 당신을 추가할 수 있도록 공유하세요.

---

## 친구 방문

1. Friendship Journal 열기
2. 온라인인 친구 선택
3. **"Visit Town"** 선택
4. 친구의 세계로 순간이동

방문 중에는 친구의 집을 둘러보고, 장식을 감상하고, 마을과 상호작용할 수 있습니다. 친구가 Co-Build 권한을 주지 않는 한 집을 수정할 수는 없습니다.

## Coffee Umbrellas

Coffee Umbrellas는 Heartopia 곳곳에 흩어져 있는 소셜 스팟입니다. 커피컵 아이콘으로 표시됩니다.

### Coffee Umbrellas에서 할 수 있는 것

- 같은 테이블에서 최대 3명과 **자유롭게 채팅**
- **퍼즐 함께 풀기** — 협동 미니게임
- **바다 낚시 퀘스트** — 함께 낚시 여행 떠나기
- 휴식하고 사진 찍기

Coffee Umbrellas는 친구 목록에 없는 플레이어와 교류하는 주요 방법입니다. 열린 테이블에 앉아 누가 합류하는지 기다려 보세요.

## Co-Building

**DG Level 8**에서 해금. 친구와 실시간으로 함께 건설하세요.

### Co-Build 시작 방법

1. Co-Build용 **전용 청사진 슬롯** 생성 (절대 메인 집 슬롯 사용 금지)
2. Friendship Journal 열고 친구 선택
3. **"Invite to Co-Build"** 선택
4. 전용 슬롯에만 건설 권한 부여

### Co-Build 안전 규칙

| Rule | Why |
| --- | --- |
| 전용 청사진 슬롯 사용 | 메인 집 우발적 손상 방지 |
| 신뢰하는 친구하고만 Co-Build | 건설 권한 악용 가능 |
| 시작 전 빌드 스크린샷 | 간편한 롤백 참조 |
| 완료 시 권한 회수 | 깔끔한 세션 종료 |

### 함께 할 수 있는 것

- 가구 동시 배치, 이동, 회전
- 복잡한 디자인을 더 빠르게 구상하고 실행
- 가구와 재료 아이디어 공유
- 서로의 변경 사항 실시간 미리보기

## Party Feature

**DG Level 10**에서 해금. 집에서 모임을 주최하세요.

### 파티 주최 방법

1. 스마트워치를 열고 Party 메뉴로 이동
2. 파티 유형 선택 (캐주얼, 테마 등)
3. Friendship Journal에서 친구 초대
4. 친구가 당신의 집으로 순간이동

### 파티 활동

- 채팅과 사교
- 최신 집 빌드 자랑
- 악기가 있다면 음악 연주
- **Tea Party** 주최 — DG 레벨 불필요, 집에서 주최 가능

## Sea Fishing with Friends

**DG Level 7**에서 해금. Bill의 배가 당신과 파티를 바다로 데려갑니다.

1. Fishing Village 부두에서 **Bill**과 대화
2. **Sea Fishing Event** 선택
3. 파티에서 친구 초대
4. 세션당 **2,000 Gold** (반복 시 3,000)
5. 함께 희귀 고가치 물고기 포획
6. 그룹 목표 완료 시 Gatcha Eggs 획득

## 크로스 플랫폼 플레이

Heartopia는 모바일과 PC 간 **완전한 크로스 플랫폼 플레이**를 지원합니다:

| Platform | How to Play |
| --- | --- |
| **iOS** | App Store |
| **Android** | Google Play |
| **PC (Steam)** | Steam (무료 플레이) |

### 크로스 플랫폼 정보

- **모바일과 PC 플레이어가 같은 서버를 공유** — 기기에 상관없이 누구와든 플레이 가능
- **진행 데이터가 기기 간 동기화** — PC에서 로그인하고 모바일에서 이어서 플레이, 모든 진행 상황이 저장됨
- **Steam은 중국 지역 잠금** — 중국 IP는 Steam에서 Heartopia에 접근할 수 없습니다. 대신 모바일 버전을 사용하세요.
- 동일한 계정으로 모든 기기에서 로그인

## 채팅 시스템

### 채팅 작동 방식

- 채팅은 **확인된 친구끼리만** 활성화
- 공개 글로벌 채팅 없음 — 낯선 사람의 메시지가 보이지 않음
- Coffee Umbrellas에서 최대 3명과 채팅 가능
- Co-Build 중에는 항상 건설 파트너와 채팅 가능

### 감정 표현

- **DG Level 3**에서 해금
- **Annie**(기본 감정 표현)와 이벤트 상점(한정 감정 표현)에서 구매
- 타이핑 없이 감정 표현으로 소통
- 2026년 3월 업데이트에서 **"tap to get attention"** 기능 추가

## 멀티플레이어 에티켓

1. **방문 전에 물어보기** — 누군가의 마을에 가기 전 짧은 메시지
2. **허락 없이 만지지 않기** — 초대받지 않았다면 다른 플레이어의 가구나 작물과 절대 상호작용하지 마세요
3. **Co-Build 경계 존중** — 건설 권한을 받았다면 합의된 범위를 지키세요
4. **호스트에게 보답** — 멋진 파티를 열어준 사람에게 작은 선물 보내기
5. **Coffee Umbrellas는 공공 공간** — 앉는 낯선 사람에게 친절하게 대하세요
