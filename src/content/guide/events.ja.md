Heartopiaでは、限定衣装、家具、エモート、レア通貨などの限定報酬が得られる定期開催の期間限定イベントが実施されます。このガイドでは、ローンチから2026年5月までのすべての主要イベントを、最も複雑なものの攻略法を含めて解説します。

---

## Winter Frost Season

**開催期間**: 2026年1月31日 〜 3月13日  
**必要条件**: DGレベル7

最初の大型シーズンイベント。島全体が雪に覆われ、Onsen Mountainがウィンタースポーツの拠点に変わり、限定の冬レシピが利用可能になります。

### アクティビティ

| アクティビティ | 場所 | 詳細 |
| --- | --- | --- |
| Ice Skating | Onsen Mountain（北） | タイミングよくボタンを押してCombo Energyを構築し、印象的な連続技を決める。2026年3月に2人スケート追加。 |
| Snow Slides | Onsen Mountain | 山を滑り降りるスノースライド |
| Snowman Building | 街中 | 雪だるまを作って報酬を獲得 |
| Winter Recipes | キッチン | 季節限定料理レシピ（Frosted Pancake、Iced Drink） |
| Frostspore Infection Quest | 沿岸エリア | 沿岸エリアを調査し、NPCやプレイヤーからスタンプを集める |

## Snow Concert

**開催期間**: Winter Frost Season中  
**場所**: Onsen Mountain山頂（Old Sea付近）— コンサート会場へのポータルが開く  
**主催**: Atara

### スケジュール

| 期間 | 開催時間 |
| --- | --- |
| オープニングウィーク | 毎日ゲーム内時間19:00 |
| 週末（2月7日から） | 土曜・日曜 18:10 CT |
| クロージングウィーク | 毎日ゲーム内時間19:00 |

所要時間：約1時間50分、10分ごとに音楽と花火。

### コンサートでのアクティビティ

- コンサートグッズ（グロースティック）の購入
- テーブルでの友達とのチャット
- リンクでのアイススケート
- コンサートバブルの収集（スパーク、バトン、楽譜、ゴールド、Fashionwave Tokens）
- 「フリーミュージック」セグメントでの自分の音楽演奏

### コンサート報酬

個人およびチーム目標の達成で**Penguin Gacha**アイテムを獲得。コンサートにはNPCミュージシャン**Beau、Muffy、K.K. Slider**も登場します。

## Wonder Egg Hunt（冬）

**開催期間**: Winter Frost Season中  
**場所**: Onsen Mountain

### 遊び方

1. **Eric**と話してWinter Frostクエストを開始
2. Onsen Mountainで**Naughty**と話してエッグハントを解放
3. **合計16個の卵** — 毎日午前6:00に新しい卵が1つ出現
4. 卵はフェンス、像、イグルーの中、スタンドの後ろ、岩の上などに隠されている
5. 前日に取り逃した卵は**後から回収可能**

### エッグハント報酬

卵1個につき：**Lucky Stars 5個 + Fashionwave Tokens 300個**（確率ベースの報酬）。

## Fashionwave

**開催期間**: Winter Frost中および以降も継続  
**場所**: Onsen Mountain（Azureのショップ）

FashionwaveはHeartopiaの定期開催ファッションチャレンジシステムです：

- 毎日のログイン、週刊チャレンジ、フェスティバルタスクで**Fashionwave Tokens**を獲得
- トークンで限定衣装、**Fashionwave Penguin Set**、乗り物、ポーズ、家具と交換
- **Exhibition Passes**でプレミアム報酬ティアを解放
- コミュニティ投票による週刊ファッションチャレンジ

### Fashionwave報酬

- 限定冬衣装とアクセサリー
- Fashionwave Penguin Set
- 乗り物とポーズ
- 建築素材
- Steel Tongue Drum楽器スキン

## Beaver Build Challenge

**開催期間**: 2026年4月25日 〜 6月20日  
**必要条件**: DGレベル12、Home Evaluation Score 1,500以上

チームベースの建築コンペティション：

### 遊び方

| フェーズ | アクティビティ |
| --- | --- |
| **募集** | 最大4人のプレイヤーでチームを結成 |
| **創作期間** | チャレンジテーマに沿って建築 |
| **展示** | コミュニティ向けにビルドを提出 |
| **選考と報酬** | 勝者が選ばれ、報酬が配布 |

### イベント通貨

**Fresh Sprouts** — イベント中に獲得し、限定の建築素材や装飾と交換。

### ヒント

- 補完的な建築スタイルを持つプレイヤーとチームを組む
- チャレンジ用のビルドには専用の設計図スロットを使う
- Discordまたはゲーム内チャットで連携
- 建築前にテーマ要件をよく確認する

## Aurora Banquet Quest

Winter Frost中に解放されるイベントクエスト：

1. クエスト依頼者と話す
2. **Iced Drink**と**Frosted Pancake**（冬のレシピ）を作成
3. バンケットで提供
4. 報酬には限定冬装飾が含まれる

### Frosted Pancakeレシピ

Winter Frost中にレシピブックで正確な材料を確認してください。冬のレシピはシーズン中のみ調理可能です。

## イベントを逃さないために

1. **Eventsページを毎日チェック** — スマートウォッチの惑星アイコンをタップ
2. **メールボックスを確認** — イベント告知は手紙で届く
3. **Heartopia公式SNSをフォロー** — Twitter/X、Facebook、Discord
4. **天気予報をチェック** — 一部のイベントは特定の天候（流星群、虹）に連動
5. **DGレベルを最新に保つ** — ほとんどのイベントにはDGレベル要件がある
