家の建築と装飾はHeartopiaで最も奥深いシステムの一つです。このガイドでは、チュートリアルでは触れられない高度なテクニック — 設計図のロードアウト、安全なCo-Build、DIY家具ハック、壁紙の裏技を紹介します。

## 設計図システム（DGレベル8）

設計図は基本的に**家のセーブスロット**です。家の複数バージョンを保存し、何も壊すことなく瞬時に切り替えられます。

### 設計図の仕組み

- スマートウォッチアプリを通じて**DGレベル8**で解放
- 各設計図スロットには**家のレイアウトの完全なコピー**が保存される
- ビルド間を自由に切り替え — 何も失われない
- 設計図モードでは**無制限の家具と装飾**で自由に実験可能

### おすすめの設計図設定

| スロット | 用途 |
| --- | --- |
| スロット1 | メインの家のビルド |
| スロット2 | 代替デザイン（例：季節テーマ） |
| スロット3 | **専用テストスロット** — ここで自由に実験 |
| スロット4 | **Co-Buildスロット** — 友達の権限はここだけに |

### 設計図の安全ルール

**メインの家のスロットでは絶対に友達に建築権限を与えないでください。** 専用の使い捨て設計図スロットをCo-Build用に用意し、友達が誤って（あるいは故意に）本物の家を壊せないようにしましょう。

## Co-Building（DGレベル8）

DGレベル8で設計図と同時に解放されます。別のプレイヤーを招待して**リアルタイム**で一緒に建築できます。

### Co-Buildの始め方

1. Co-Build用に専用の設計図スロットを作成
2. Friendship Journalから友達を招待
3. 建築権限を**そのスロットのみ**に付与
4. 両方のプレイヤーが同時に家具の配置、移動、回転を行える

### Co-Build安全チェックリスト

- 使い捨て設計図スロットを使う — メインは絶対に使わない
- 信頼できる友達とのみCo-Buildする
- 開始前にビルドのスクリーンショットを撮る
- 権限はいつでも取り消せる

## カスタム家具ハック

高価な家具を買う代わりに、基本素材で作れるのに気づいていますか？これらのDIYテクニックで何千ゴールドも節約できます。

### DIYシェルフ

**塗装された床パネル**を壁に対して平行に積み重ねます。複数のパネルで多段シェルフを作成 — コレクションの展示に最適です。

### DIYダイニングテーブル

1. **2本の柱**を最小の高さに縮小（テーブルの脚）
2. テーブルを置きたい場所に配置
3. 天板として**大理石または木製の床パネル3枚**で覆う

### DIYキッチンアイランド

**ハーフウォール＋床パネル**のみで構築。Bob's Shopでキッチンアイランドを買うよりはるかに安く、自分の色と素材で完全にカスタマイズ可能です。

### DIY展示台

屋内に**短い柱**を配置して装飾品、トロフィー、レアコレクションを高く見せて展示します。部屋に合わせて塗装しましょう。

## 壁紙の裏技

### 壁紙パターンのミックス＆マッチ

異なる壁紙のパターンを同じ壁に組み合わせることができます：

1. **低い壁**セクションを配置
2. その上に**2つのハーフウォール**を追加
3. 各セクションを**異なる壁紙**で塗装
4. これで1つの壁紙の幅木と別の壁紙のパターンを同時に使える

### 塗り直さずに壁を拡張

**オーバーヘッドモード**で、既存の壁や床タイルをドラッグして拡張 — 塗装とパターンは自動的に引き継がれます。手動で再適用する必要はありません。

## 屋内での外装素材活用

### 作り付けアクアリウム展示

**四角い木製の窓**を使って見事な作り付けアクアリウムを作成：

1. 展示したい場所に**低い壁の台**を構築
2. 台の上に水槽を配置
3. 壁パネルで囲む
4. 両面を**木製の四角い窓**でフレーム
5. 結果として壁に組み込まれたシースルー水槽が完成

## Builder Tabletのヒント

Builder Tablet（スマートウォッチからアクセス）は精密建築ツールです：

- **グリッドビュー** — ピクセル単位で正確な家具配置
- **埋まったアイテムの回収** — 壁の中に埋まって手動でクリックできない家具を回収
- **回転と移動** — 家具を持ち上げずに操作
- **環境調整ツール** — 異なる照明、天候、時間帯でビルドをプレビュー。窓は照明によって色が変わるので、確定前にチェックを。

## Home Plotsの拡張

小さな区画から始まり、DGレベルが上がるにつれて拡張されます：

| DGレベル | Home Plots |
| --- | --- |
| 開始時 | 1 |
| レベル4 | 4 |
| レベル6 | 5 |
| レベル8 | 6 |
| レベル11 | 7 |
| レベル14 | 8 |
| レベル16 | 9 |

### より多くの区画を解放する方法

- 毎日のResident RequestsでDGをレベルアップ
- 必要なDGレベルに達すると区画拡張は自動的
- 新しい区画ごとに追加の建物を建てたり庭を拡張できる
