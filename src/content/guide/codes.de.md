Codes sind kostenlose Geschenk-Codes, die von XD Entertainment veröffentlicht werden und dir In-Game-Belohnungen geben — Moonlight Crystals, Wishing Stars, Gold, Reparatursets und seltene Materialien. Diese Seite wird regelmäßig mit allen aktiven Codes aktualisiert.

## So löst du Codes ein

1. Starte **Heartopia** und betrete einen Live-Server
2. Öffne deine **Uhr** — tippe auf das Uhr-Symbol oben rechts (PC: Drücke **Tab**)
3. Gehe zu **Einstellungen** aus dem Uhren-Menü
4. Tippe auf **„Redeem Code"** unten rechts auf dem Einstellungsbildschirm
5. Gib einen Code genauso ein, wie er angezeigt wird (Kopieren und Einfügen ist am sichersten, um Tippfehler zu vermeiden)
6. Drücke **Redeem** — eine Erfolgsmeldung erscheint
7. Hole deine Belohnungen aus dem **Briefkasten** vor deinem Haus ab

> **Wichtig:** Codes sind nicht case-sensitiv, müssen aber genau eingegeben werden. Jeder Code kann nur **einmal pro Konto** verwendet werden. Einige Codes verfallen ohne Vorwarnung — löse alles so schnell wie möglich ein.

## Tipps zum Einlösen von Codes

- **Achte auf Leerzeichen am Ende** beim Einfügen — ein Leerzeichen am Ende kann dazu führen, dass der Code fehlschlägt
- **Sofort einlösen** — die meisten Codes sind zeitlich begrenzt. Einige verfallen innerhalb weniger Tage nach Veröffentlichung.
- **Folge den offiziellen Heartopia-Social-Media-Kanälen** (Twitter/X, Facebook, Discord) für neue Code-Drops
- **Codes sind einmalig pro Konto** — du kannst einen Code nicht auf demselben Profil wiederverwenden
- **Einige Codes könnten abgelaufen sein** — wenn ein Code nicht funktioniert, ist er möglicherweise abgelaufen, seit diese Seite zuletzt aktualisiert wurde. Überprüfe das Datum unten.
