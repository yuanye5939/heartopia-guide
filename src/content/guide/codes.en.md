Redeem codes are free gift codes released by XD Entertainment that give you in-game rewards — Moonlight Crystals, Wishing Stars, Gold, Repair Kits, and rare materials. This page is updated regularly with all active codes.

## How to Redeem Codes

1. Launch **Heartopia** and enter a live server
2. Open your **watch** — tap the clock icon in the top-right corner (PC: press **Tab**)
3. Go to **Settings** from the watch menu
4. Tap **"Redeem Code"** in the bottom-right of the Settings screen
5. Enter a code exactly as shown (copy-paste is safest to avoid typos)
6. Hit **Redeem** — a success message appears
7. Claim your rewards from the **mailbox** outside your house

> **Important:** Codes are not case-sensitive but must be entered exactly. Each code can only be used **once per account**. Some codes expire without warning — redeem everything as soon as possible.

## Tips for Redeeming Codes

- **Check for trailing spaces** when pasting — a space at the end can cause the code to fail
- **Redeem immediately** — most codes are time-limited. Some expire within days of release.
- **Follow official Heartopia social media** (Twitter/X, Facebook, Discord) for new code drops
- **Codes are one-time use per account** — you cannot reuse a code on the same profile
- **Some codes may be expired** — if a code doesn't work, it may have expired since this page was last updated. Check the date below.
