Les codes d'échange sont des codes cadeaux gratuits publiés par XD Entertainment qui vous donnent des récompenses en jeu — Moonlight Crystals, Wishing Stars, Gold, Kits de réparation et matériaux rares. Cette page est mise à jour régulièrement avec tous les codes actifs.

## Comment utiliser les codes

1. Lancez **Heartopia** et connectez-vous à un serveur
2. Ouvrez votre **montre** — appuyez sur l'icône de l'horloge en haut à droite (PC : appuyez sur **Tab**)
3. Allez dans **Paramètres** depuis le menu de la montre
4. Appuyez sur **« Redeem Code »** en bas à droite de l'écran des paramètres
5. Saisissez un code exactement comme indiqué (le copier-coller est le plus sûr pour éviter les fautes de frappe)
6. Appuyez sur **Redeem** — un message de succès apparaît
7. Récupérez vos récompenses dans la **boîte aux lettres** devant votre maison

> **Important :** Les codes ne sont pas sensibles à la casse mais doivent être saisis exactement. Chaque code ne peut être utilisé qu'**une seule fois par compte**. Certains codes expirent sans préavis — utilisez tout dès que possible.

## Conseils pour l'utilisation des codes

- **Vérifiez les espaces en fin de texte** lors du collage — un espace à la fin peut faire échouer le code
- **Utilisez-les immédiatement** — la plupart des codes sont à durée limitée. Certains expirent quelques jours après leur publication.
- **Suivez les réseaux sociaux officiels de Heartopia** (Twitter/X, Facebook, Discord) pour les nouvelles sorties de codes
- **Les codes sont à usage unique par compte** — vous ne pouvez pas réutiliser un code sur le même profil
- **Certains codes peuvent être expirés** — si un code ne fonctionne pas, il a peut-être expiré depuis la dernière mise à jour de cette page. Vérifiez la date ci-dessous.
